## About Installation Keynotion

Steps

- ```git clone```
- ```composer update```
- ```php artisan key:generate```
- ```php artisan migrate:fresh --seed```
- ```Admin login-password in AdminSeeder (login from *path (/login))```
- ```php artisan serve```
- ```php artisan ide-helper:models -M```
